# nlp_project
NLP-Driven Place Matching For Personalized Travel Itineraries

## Project Overview
This project processes a dataset containing descriptions of various tourist locations. Multiple vectorization techniques are used to transform textual descriptions into mathematical representations that can be compared with user queries. Once similarity scores are computed, the top 5 most relevant results are displayed on the interface.
---

## Tech Stack
- **Python 3.11.10** – Programming language  
- **Streamlit** – Web application framework for interactive UI  
- **NLTK / spaCy / Scikit-learn** – NLP and machine learning libraries  
- **requirements.txt** – Dependency management for virtual environment  

---

## Setup Instructions
first setup your virtual environment venv
in it install dependencies: pip install -r requirements.txt 
Final command:  streamlit run app.py
